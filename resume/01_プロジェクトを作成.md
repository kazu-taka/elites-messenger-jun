## プロジェクトを作成

`rails new`コマンドでプロジェクトを作成する。
```
$ rails new elites-messenger-repo
$ cd elites-messenger-repo
```

プロジェクトをGit管理する場合は以下のディレクトリおよびファイルを管理外にする。
```
# .gitignore

# 以下を追記
/public/uploads
/public/assets
.DS_Store
```

### 補足: gitignore

特定のファイルやディレクトリをGit管理しない場合は`.gitignore`を作成する

```
# .gitignore
# 記入例

# 「test.txt」ファイルを対象から除外
test.txt

# 拡張子が「o」や「a」のファイルを除外
*.[oa]

# exclusion.oは除外しない
!exclusion.o

# 「test」ディレクトリを除外
/test
```
